# 🎮 Final Project – Unity Game

## 📌 Overview

This project is a **Unity-based game/application** developed using **C#**. It demonstrates core game development concepts including scripting, scene management, and integration with Unity services.

The project structure includes multiple `.csproj` files and Unity configurations, indicating a complete development environment setup.

## 📂 File Descriptions

### 📄 Assembly-CSharp.csproj

Contains the main game scripts written in C#.

### 📄 FinalProject.csproj

Core project configuration file used by the IDE.

### 📄 FinalProject.sln

Solution file to open the project in Visual Studio.

### ⚙️ Unity.*.csproj

Auto-generated Unity project files for:

* Analytics
* Collaboration tools
* TextMeshPro

### ⚙️ .collabignore

Specifies files ignored by Unity Collab.

### ⚠️ .DS_Store

System file (can be ignored or removed).

## ▶️ How to Run the Project

### 1️⃣ Open in Unity

* Open **Unity Hub**
* Click **Add Project**
* Select the project folder

### 2️⃣ Open the Project

* Use a compatible Unity version (recommended: same version used during development)

### 3️⃣ Run the Game

* Open the main scene
* Click ▶️ **Play** inside Unity Editor
